# My First Logger Package

## Installation

```console
npm install logger
```

or

```console
yarn add logger
```

or

```console
pnpm install logger
```

## Basic Usage

- This Package Built In Typescript Declaration

### Typescript

```ts
import Logger from "logger";

const logger: Logger = new Logger({});

logger.info("Hello, I Am A Normal Log!");
```

### Javascript

```js
import Logger from "logger";

const logger = new Logger({});

logger.info("Hello, I Am A Normal Log!");
```
